## Build Instructions
1. Ensure JDK 17 and Android SDK are installed.
2. Install Bubblewrap CLI: `npm install -g @bubblewrap/cli`.
3. Navigate to `twa/`.
4. Run `bubblewrap build`.
5. Enter keystore password when prompted from the project owner.

## Remaining Manual Steps
1. **Deploy Asset Links:** Deploy `.well-known/assetlinks.json` to `https://packzen.in/.well-known/assetlinks.json`. This is required to remove the browser address bar in the app.
2. **Play Console Setup:** Create an application on the Google Play Console.
3. **Upload AAB:** Upload `twa/app-release-bundle.aab` to the internal testing track.
4. **App Listing Details:** Complete all store listing details (icons, screenshots, descriptions, policies).
